####---#  Play Store full description
Avec MiUI Statusbar Pro vous pouvez changer le style de votre barre de notification dans MiUI.. Nos modifications de style ne nécessitent pas le framework Xposed et n'entrent pas en confilt avec les thèmes MIUI.
Tout a commencé avec le mod "Center Clock Mod" et maintenant vous poouvez choisir parmi 7 styles differents !

Fonctionnalités :
**Crée des sauvegardes de MiuiSystemUI.apk
**Restaure des sauvegardes de MiuiSystemUI.apk
**Change le style de la barre de notification de MUI dans 7 modes differents.
**Fonctionne aussi sur des appareils non-Xiaomi tournant sous MiUI !
**Pas de framework Xposed requis !
 
Prérequis :
**Root nécessaire
**Disponible uniquement sous MIUI
